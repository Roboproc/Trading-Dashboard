☁ CLOUD SYNC

Sync your trades, journal, and watchlist across all your devices. Uses Supabase (free).

1. Go to supabase.com → create free account → New project
2. Authentication → Providers → Email → turn OFF "Confirm email"
3. SQL Editor → New query → paste & run the SQL below
4. Settings → API Keys → copy URL & anon (public) key
5. Paste below, pick a password, and click Connect

-- Run this in Supabase SQL Editor:
create table apexx_sync (
  user_id uuid references auth.users primary key,
  payload jsonb not null,
  updated_at timestamptz default now()
);
alter table apexx_sync enable row level security;
create policy "own" on apexx_sync
  using (auth.uid()=user_id)
  with check (auth.uid()=user_id);

  {"ticker":"HOOD","shares":16,"entry":69.26}
]
